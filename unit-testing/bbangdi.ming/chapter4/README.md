# 좋은 단위 테스트의 4대 요소
- 회귀 방지
  - 테스트가 가능한 많은 코드를 실행하는 것을 목표로 해야한다.
- 래팩터링 내성
  - 거짓 양성이 발생하지 않도록 테스트를 작성해야한다.
    - 거짓 양성을 줄이는 방법: 테스트를 작은 단위로 분리
- 빠른 피드백
- 유지 보수성
  - 테스트가 얼마나 이해하기 어려운가
  - 테스트가 얼마나 실행하기 어려운가
